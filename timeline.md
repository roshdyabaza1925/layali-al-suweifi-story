```

├─ فيلا زيزينيا = الحصار المخملي
└─ المشاعر: أمان مطلق، براءة كاملة

├─ مدرسة الراهبات الإيطالية
├─ تعلم اللغة الإيطالية بطلاقة
├─ الفروسية والسباحة ببراعة
├─ بدايات وعي بـ الجمال والاختلاف
└─ بناء "الهيبة الطبيعية"
```
